- 作用  
对序列进行过滤操作
- 输入输出模型  
输入：过滤函数和序列   
输出：Iterator
- 过滤方法  
将函数作用于序列中的每一个元素，如果结果为 True 则保留该元素，为 False 则舍弃改元素
- 非常值得一看的代码测试
```python
>>> def is_odd(n):
...     return n % 2 ==1
...
>>> is_odd(2)
False
>>> is_odd(3)
True
>>> l = [1,2,3,4,5]
>>> f = filter(is_odd,l)
>>> f
<filter object at 0x00000000007326D8>
>>> from collections import Iterator
>>> isinstance(f,Iterator)
True
>>> next(f)
1
>>> list(f)
[3, 5]
>>> l
[1, 2, 3, 4, 5]
>>>
```
